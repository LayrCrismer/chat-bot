# Консольный Чат-бот на C++
✨ ## Основные возможности<br>
**Интерактивный диалог**: Бот ищет ответы в загруженной базе данных.<br>
**Админ-панель**: Позволяет добавлять, удалять и редактировать вопросы и ответы прямо во время работы программы.<br>
**Управление базами данных**: Загрузка существующих файлов .txt.<br>
**Создание новых баз данных**.<br>
**Автоматическое сохранение изменений**.<br>
**Кастомизация**: Возможность изменить имя бота через админ-интерфейс (имя сохраняется в первой строке файла базы).<br>
**Надежность**: Обработка некорректного ввода пользователя (защита от бесконечных циклов при вводе букв вместо чисел).<br>

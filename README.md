# Консольный Чат-бот на C++
✨ ## Основные возможности\n
**Интерактивный диалог**: Бот ищет ответы в загруженной базе данных.
**Админ-панель**: Позволяет добавлять, удалять и редактировать вопросы и ответы прямо во время работы программы.
**Управление базами данных**: Загрузка существующих файлов .txt.
**Создание новых баз данных**.
**Автоматическое сохранение изменений**.
**Кастомизация**: Возможность изменить имя бота через админ-интерфейс (имя сохраняется в первой строке файла базы).
**Надежность**: Обработка некорректного ввода пользователя (защита от бесконечных циклов при вводе букв вместо чисел).
